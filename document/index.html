Installation

Follow the steps below to setup your site template:

  1. Unzip the downloaded package and open the Xaya Main folder to find the template files. You will need to upload these files to your hosting web server using FTP or localhost in order to use it on your website.
  2. Pay Attention! You need a NodeJs host, or Server to run this template. Read this guide for more information

If you want to open this template on your personal computer, follow the guide below

  1. Unzip the downloaded package and open the Xaya Main folder to find the template files.
  2. Press Alt + D to focus the adress bar. type cmd and press enter.
  3. type npm i --force Then the packages will start downloading and you have to wait a little.
  4. Then type npm start to run the React app.
  5. done! enjoy it :)

Color Schemes

You can change the color scheme of your website in an instant. We made 4 color mods

  • orange-mode - Orange Color
  • red-mode - Red Color
  • blue-mode - Blue Color
  • purple-mode - Purple Color

Blue mode is the default.
you can also select your color in home page (at the top right)

If you want to change the default color. Follow the instructions below :

  1. Open the file : src --> config --> settings.sjon
  2. In line 3, set the value of the defaultColorMode property with your desired color mode. (Only predefined modes)

If you don't know about json files and properties. Read this guide for more information

Follow the instructions below to hide the color picker on the main page of the template

  1. Open the file : src --> config --> settings.sjon
  2. In line 2, set the value of the showColorpicker property to false

Theme Customization

We have include a Custom SCSS File in the Main Folder so that you can better handle your Customizations for New Styles or Overwriting Default Theme Styles. Simply add all your Custom CSS Codes to : Xaya Main --> global.scss

If you want to add style to your desired component. Add the desired class in the global.scss file and in the component file, style the element you want to change.

Because in this project we used tailwind css, to change the class of some elements, you need to edit tailwind css classes in the desired component.

Change colors of color modes

  1. Open the file : Xaya Main --> global.scss
  2. In line 5 to 30 , You can change the hexadecimal value of each color mode

If you don't know about hexadecimal colors, Read this guide for more information

Skills Settings

Remove, Add or edit Skills

  1. Open the file : Xaya Main --> src --> components --> home --> Skills --> SkillsSection --> SkillsData.js
  2. In this file, you can change the content of the skills.
    To add skills to the SkillsData array, you must add an object with the following format to its indexes : { name: 'Figma', progress: 100, }

Set the value of the name property to the name of the skill and the value of the progress property to the mastery value

Change Skills Description

  1. Open the file : content.json
  2. On line 9, set the value of the skills-description property to whatever you want

Remove, Add or edit Experience info cards

  1. Open the file : Xaya Main --> src --> components --> home --> Skills --> ExperienceSection --> ExperienceData.js
  2. In this file, you can change the content of the skills.
    To add skills to the ExperienceData array, you must add an object with the following format to its indexes :
    {
     companyName: 'Google',
     date: '(2016 - 2019)',
     jobTitle: 'Front-End Developer',
     description: ' Adipisicing Lorem ipsum dolor sit amet,
     consectetur elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.',
    },

Portfolios Settings

Change images in the slider

  1. Add your image in this folder : Xaya Main --> src --> assets --> images
  2. Open the file : Xaya Main --> src --> components --> home --> CreativeWorks --> CreativeWorksImages.js
  3. Then import it into this file. for example : import img4 from '../../../assets/images/slide4.jpg'
  4. In this file, you can change the slider images.
    To add image to the slider array, you must add an object with the following format to its indexes :
    {
     src: img1,
     alt: 'alt',
    },
